Tentang Saya

Halo, saya Rizki Suputro, seorang fresh graduate SMK jurusan Teknik Komputer & Jaringan (TKJ) yang memiliki minat besar di dunia teknologi dan pengembangan web.

Selama masa studi, saya terbiasa mempelajari dasar-dasar jaringan komputer, perakitan hardware, troubleshooting, hingga pengelolaan sistem operasi. Selain itu, saya juga mulai mengembangkan kemampuan di bidang pemrograman web sebagai bekal untuk karier saya ke depan. Saya senang mempelajari hal-hal baru, beradaptasi dengan cepat, dan mampu bekerja baik secara individu maupun dalam tim.

Bagi saya, teknologi bukan hanya tentang perangkat keras dan lunak, tetapi juga tentang bagaimana menciptakan solusi yang bermanfaat untuk orang lain.

Skills

HTML

Bahasa dasar untuk membangun struktur halaman web dengan elemen semantik yang jelas, terorganisir, dan mudah dipahami oleh mesin pencari modern.

CSS

Digunakan untuk mengatur tampilan halaman web agar lebih rapi, menarik, konsisten, serta responsif pada berbagai ukuran layar berbeda.

JAVASCRIPT

Bahasa pemrograman populer yang menambahkan interaktivitas, animasi, dan logika dinamis sehingga halaman web terasa lebih hidup dan responsif.

BOOSTRAP

Utility-first CSS framework yang fleksibel, ringan, modern, serta mendukung pengembangan cepat dengan class bawaan yang dapat dikustomisasi penuh.

Ui Design

Proses merancang antarmuka pengguna yang estetis, fungsional, konsisten, dan fokus pada pengalaman pengguna agar aplikasi lebih nyaman digunakan.
